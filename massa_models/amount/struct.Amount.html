pub struct Amount(_);
Expand description

A structure representing a decimal Amount of coins with safe operations this allows ensuring that there is never an uncontrolled overflow or precision loss while providing a convenient decimal interface for users The underlying u64 raw representation if a fixed-point value with factor AMOUNT_DECIMAL_FACTOR The minimal value is 0 and the maximal value is 18446744073.709551615
